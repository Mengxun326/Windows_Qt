Windows_Qt/FaceExtraction
==============================

人脸提取训练及识别
------------------------------

### opencv-3.4.1-output为已经编译好的MinGW版本

### 主要功能:
	  1、人脸提取及保存;  
          2、模型训练;  
          3、人脸识别;  

### 需改进的地方:
              1、可重复提取人脸到指定文件夹;  
              2、训练特定路径的人脸图片文件夹;  
              3、可将训练信息存储到同一模型中;  
              4、可识别多个人脸;
