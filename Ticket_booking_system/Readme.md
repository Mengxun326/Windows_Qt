# Windows_Qt/Ticket_booking_system  
===============================  

基于Qt开发的机票预订系统:   
_______________________________   

###########部署准备  
1.此版本基于Qt5.6.1进行开发  
2.首先建立flight_book_system数据库，然后运行database文件夹中的sql文件  
3.在项目database.h中修改本机数据库相关信息即可连接数据库
